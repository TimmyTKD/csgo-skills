Some skills you can use to get "better" at CS:GO

- Advanced Eyesight

  - Resistance to Flashing Lights
- Better Aim
  - Ability to Control Recoil 
  - Precise Mouse Movement
- Faster Reflexes
- Professional Parkour Skills
